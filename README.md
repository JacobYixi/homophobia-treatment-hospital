# 博爱恐同症矫治医院

> 这是一个讽刺性艺术项目。

本项目以虚构医疗机构官网的形式，将"恐同症"框架化为需要临床矫治的"疾病"，通过反讽手法反思社会对性少数群体的偏见与歧视。

网站包含首页、诊疗项目、专家团队、康复案例、在线自测、在线预约、科普知识、医院动态、常见问题等模块，以极度正式、权威的医疗机构口吻进行叙事。

## 声明

- **本项目纯属艺术创作与讽刺，不具备任何医学诊断或治疗效力。**
- **我们不认为同性恋是疾病。需要被"矫治"的从来只有恐同症。**
- 项目中的所有"疗法"均为对历史上真实存在过的强制"矫正治疗"的隐喻与反讽。
- 项目内的在线自测问卷仅供娱乐参考，结果不具备临床意义。

## 本地运行

```bash
pnpm install
pnpm dev
```

Next.js 16 + React 19 + TypeScript + Tailwind CSS
